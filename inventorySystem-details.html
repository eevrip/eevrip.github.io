Inventory System

Recently I was playing Subnautica Below Zero and it inspired me to develop my own inventory system with the same mechanics as that game.

Design

The inventory system is designed to be sorted in a custom way and each slot is not stackable. The user can assign objects to the toolbar either from their backpack inventory or the toolbar. The user can also use storage to storage their items and retrieve them in the future. I used Scriptable Objects to create different kind of items divided by categories such as resource, consumable, equipment and tool. Each category has its own features but inherit from the same Scriptable object.
